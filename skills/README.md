# Project Skills

This directory contains Cursor-compatible skills for the Hooras frontend project.

## Available Skills

| Skill | Path | When to use |
|-------|------|-------------|
| Frontend Design | [frontend-design/SKILL.md](./frontend-design/SKILL.md) | All UI/UX work — tokens, components, layout, light/dark theming |

## Adding New Skills

1. Create a new folder under `skills/` with a `SKILL.md` file
2. Include YAML frontmatter with `name` and `description` fields
3. Write all content in **English**
4. Update this README with the new skill entry
